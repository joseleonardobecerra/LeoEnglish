# EduEnglish - Plataforma Gamificada de Aprendizaje

Una aplicación interactiva para aprender inglés básico mediante metodologías activas (Conceptualizar, Practicar y Evaluar). Construida con React, Vite y Tailwind CSS.

## Características
- **Dashboard Gamificado:** Seguimiento de XP (Puntos de Experiencia), rachas diarias y progreso en el Marco Europeo (Grammar, Vocabulary, Reading, Writing).
- **Módulos de Teoría y Práctica:** Preposiciones, Números, Verbos, Vocabulario y Textos de lectura.
- **Juegos Interactivos:** Conectar palabras (Match Game), ordenar frases (Word Order) y Flashcards.

## Instalación y Uso Local
1. Clona este repositorio: `git clone <tu-url-del-repo>`
2. Instala las dependencias: `npm install`
3. Ejecuta el servidor de desarrollo: `npm run dev`
